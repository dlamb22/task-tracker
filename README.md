# Task-Tracker

Task-Tracker was created using ReactJS

## Installation

Using npm:

```bash
git clone https://github.com/dlamb22/task-tracker.git
cd task-tracker
npm install
```

## Usage

Once it has finished installing all of the packages, run this code:

```bash
npm start
```
